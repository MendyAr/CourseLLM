# CourseLLM User Stories

## Student Stories

### Learning Interaction
1. As a student, I want to ask questions about course materials so that I can better understand complex concepts
2. As a student, I want the AI to guide me through concepts rather than give direct answers so that I develop deeper understanding
3. As a student, I want to understand why I'm learning specific topics so that I stay motivated
4. As a student, I want to track my progress through the course material so that I know where I stand

### Technical Support
1. As a student, I want help setting up development environments so that I can focus on learning
2. As a student, I want assistance debugging technical issues so that I don't get stuck on tooling
3. As a student, I want guidance on assignment submission so that I follow correct procedures

### Resource Access
1. As a student, I want access to relevant additional resources so that I can explore topics deeper
2. As a student, I want personalized practice exercises so that I can validate my understanding
3. As a student, I want to know what topics I should review so that I can fill knowledge gaps

## Teacher Stories

### Content Management
1. As a teacher, I want to upload course materials so that students can access them through the AI
2. As a teacher, I want to define learning trajectories so that students follow an optimal learning path
3. As a teacher, I want to review content consistency so that there are no contradictions

### Student Monitoring
1. As a teacher, I want to track student progress so that I can identify who needs help
2. As a teacher, I want to see common problem areas so that I can adjust my teaching
3. As a teacher, I want to verify student understanding so that I know the material is being learned

### Assignment Management
1. As a teacher, I want to create and validate assignments so that they test appropriate knowledge
2. As a teacher, I want to ensure assignments are LLM-resilient so that they promote genuine learning
3. As a teacher, I want to detect potential cheating so that I can maintain academic integrity
