# CourseLLM Glossary

## Core Concepts

### Learning Terms

- **Socratic Learning**
  - A method of teaching that uses questions to guide students toward understanding
  - Emphasizes critical thinking and self-discovery over direct answers

- **Learning Trajectory**
  - A planned sequence of topics and concepts
  - Ordered by dependencies and complexity
  - Customized based on student progress

- **Mastery Level**
  - Measure of student understanding of a specific topic
  - Determined through interactions and assessments
  - Used to guide learning path progression

- **RAG (Retrieval-Augmented Generation)**
  - AI technique that combines knowledge retrieval with text generation
  - Used to ensure AI responses are grounded in course materials

### System Components

- **Course Material**
  - Official content provided by teachers
  - Includes lectures, assignments, exercises, and resources
  - Serves as the authoritative source for AI responses

- **Learning Assistant**
  - AI-powered chatbot that helps students learn
  - Uses Socratic questioning methods
  - Maintains focus on course material scope

- **Progress Tracking**
  - System for monitoring student advancement
  - Records interactions and assessment outcomes
  - Identifies knowledge gaps and learning patterns

### User Roles

- **Student**
  - Primary learner in the system
  - Interacts with AI learning assistant
  - Follows personalized learning trajectories

- **Teacher**
  - Course content provider and manager
  - Defines learning objectives and trajectories
  - Monitors student progress and performance

- **Administrator**
  - System manager with full access
  - Handles user management and permissions
  - Ensures system integrity and security

### Technical Terms

- **Content Indexing**
  - Process of organizing course materials for RAG
  - Creates searchable knowledge base
  - Maintains relationships between topics

- **Knowledge Graph**
  - Network of course concepts and their relationships
  - Used to understand topic dependencies
  - Guides learning trajectory creation

- **LLM-Resilient**
  - Designed to prevent circumvention using AI tools
  - Focuses on demonstrating understanding
  - Requires application of knowledge

### Assessment Terms

- **Formative Assessment**
  - Ongoing evaluation during learning process
  - Used to guide teaching and learning
  - Includes practice exercises and quizzes

- **Summative Assessment**
  - Final evaluation of learning outcomes
  - Measures achievement of course objectives
  - Includes final exams and projects

- **Cheating Detection**
  - Methods to identify academic dishonesty
  - Includes pattern analysis and verification
  - Ensures learning integrity
